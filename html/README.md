# Workflows

Markdown SOPs defining objectives, steps, and edge cases.

## Workflow template
Each workflow should include:
- **Objective**: What this workflow accomplishes
- **Inputs**: What's needed to run it
- **Tools**: Which scripts in `tools/` to use
- **Steps**: Ordered instructions
- **Outputs**: What to expect when it succeeds
- **Edge cases**: Known failure modes and how to handle them
